# Online Examination System

## Project Overview

The Online Examination System is a Java Swing-based desktop application that allows students to log in, update their profile, attend an online multiple-choice examination, and view their results after submitting the exam.

## Objective

To develop a simple GUI-based examination system that provides a secure login, timed examination, automatic evaluation, and result generation.

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)

## Features

- Student Login
- Profile Update
- Multiple Choice Questions (MCQs)
- Previous and Next Navigation
- Countdown Timer
- Automatic Submission when Time Ends
- Manual Submit with Confirmation Dialog
- Result Screen
- Logout Option
- Window Close Confirmation

## How to Run

1. Open the project in NetBeans, Eclipse, or IntelliJ IDEA.
2. Compile the project.
3. Run `Main.java`.
4. Login using the default username and password.
5. Update your profile if needed.
6. Click **Start Exam**.
7. Answer the questions using the Next and Previous buttons.
8. Submit the exam manually or wait for the timer to finish.
9. View your score on the result screen.
10. Click Logout to return to the login page.
